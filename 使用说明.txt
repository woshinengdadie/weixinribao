微信工作日报助手 - 使用说明
================================

📦 给客户的安装包
-----------------
分发的安装包内只有 WeChatWorkAgent.exe 和相关文件。


🔐 客户首次使用（激活 + 配置）
-------------------------------

1. 将本目录下的 wx_key 文件夹，复制到非 C 盘的根目录下
   （例如：D:\wx_key\ 或 E:\wx_key\）

2. 进入复制后的 wx_key 文件夹，双击运行 vc_redist.x64.exe
   根据安装向导提示，完成 Visual C++ 运行库的安装

3. 在同目录下运行 wx_key.exe，根据程序提示操作
   （程序会自动关闭正在运行的微信，并重启微信）
   正常登录微信即可，随后工具会显示获取到的密钥
   复制显示的 passphrase（一串十六进制字符串），妥善保存

4. 回到本目录，双击运行 WeChatWorkAgent.exe
   首次启动会弹出激活码输入框，请输入管理员提供的激活码完成激活
   （格式: XXXX-XXXX-XXXX-XXXX）

5. 进入软件主界面后，点击左侧菜单「密钥配置」
   在「手动密钥配置」区域，将上一步保存的 passphrase 粘贴到输入框
   点击「配置密钥」按钮，等待提示「密钥配置成功」

6. 侧栏「基本设置」→ 填写你的微信昵称（必填）

7. （可选）填写 LLM API Key 开启 AI 智能总结


📋 日常使用
-----------
- 首页点击「▶ 手动运行」→ 生成今天的日报
- 可设定起始时间，只分析指定范围的消息
- 点「⏸ 自动运行」→ 每天定时自动生成
- 生成的文件：output/手动运行_日期_时间/


🔑 主要功能
-----------
🏠 首页        → 操作按钮 + 统计卡片 + 运行日志
⚙️ 基本设置    → 微信昵称 / API Key / 输出路径
🔑 密钥配置    → 微信 4.1+ passphrase 配置
⏰ 定时任务    → 每天定时 / 间隔运行
📊 周报        → 选择周次生成周报
🔍 会话分析    → 选定群聊/私聊全方面深度分析
📝 规则逻辑    → 自定义 AI 分析规则
🚫 屏蔽列表    → 精确匹配屏蔽指定会话


⚠️ 注意事项
-----------
- 本程序需要微信桌面版正在运行并已登录
- 所有数据均在本地处理，不会上传云端
- 微信 4.1+ 版本需使用 wx_key 手动配置密钥
- 公众号消息自动屏蔽，无需手动设置


🛠️ 构建工具（不随安装包分发）
-------------------------------
build_exe.bat       → 打包为 EXE 安装包
build_installer.bat → 生成 Inno Setup 安装程序
